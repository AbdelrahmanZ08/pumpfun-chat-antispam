Manage Trigger Words
Instructions
Add trigger words/phrases (one per line). Messages containing these will be processed by the extension.
Pro Tips
• Use lowercase for better matching
• Include @ for username patterns
• Use @*rug* for regex patterns
• Test with "Highlight" mode first
• Use "exact:" for exact matching
• Changes apply immediately
Examples
stoprugs
@*rug*
exact:full bundled dont buy
